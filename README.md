En este repositorio encontrara una libreria desarrollada en Java, cuyo objetivo es servir de calculadora de numeros complejos.
## Utilidades

### Version 1

* Suma de numeros complejos y matrices de estos.
* Producto de numeros complejos y matrices de estos.
* Resta de numeros complejos y matrices de estos.
* División de numeros complejos y matrices de estos.
* Módulo de un número complejo.
* Conjugado de un número complejo.
* Retornar la fase de un número complejo.

### Version 2

* Multiplicación escalar de matrices complejas.
* Matriz transpuesta.
* Matriz conjugada.
* Matriz adjunta.
* Función para calcular la "acción" de una matriz sobre un vector.
* Norma de matrices.
* Distancia entrematrices.
* Revisar si una matriz es unitaria.
* Revisar si una matriz es Hermitian.
* Producto tensor.

### Version 3

* Experimentos de la canicas con coeficiente complejos.
* Experimentos de las múltiples rendijas clásico probabilístico, con más de dos rendijas.
* Experimento de las múltiples rendijas cuántico.

### Version 4

* Calcular la probabilidad de encontrarlo en una posición en particular.
* Buscar la probabilidad de transitar entre kets.
